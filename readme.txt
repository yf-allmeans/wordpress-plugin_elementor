== Description ==

Are you looking for a way to organize your tasks, then this todo list plugin is the one for you!
This is a simple todolist plugin where you can list out your tasks and view them in an organized manner so you won't lose track of your things to do!

Once activated, you can now manage your tasks effectively in your admin panel, and even display them using an elementor widget in your page with filter settings!

== Screenshots ==
